# Simple Web Calculator

A lightweight, fully functional web-based calculator. This project features a frontend calculator interface powered by HTML, CSS, and client-side JavaScript, which is served by a backend Node.js server.

## Features

*   **Clean UI:** Modern, responsive layout built with CSS Grid.
*   **Core Math Operations:** Supports addition, subtraction, multiplication, and division.
*   **Custom Server:** Built-in lightweight Node.js server to host the application locally without external dependencies.

## Tech Stack

*   **Frontend:** HTML5, CSS3, JavaScript (Browser)
*   **Backend:** Node.js (Core modules: `http`, `fs`, `path`)
